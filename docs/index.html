Windows right-click EXIF copier

Exif Copy Tool

写真を右クリックしてフォーマットを選ぶだけで、投稿用のEXIF情報をそのままコピーできるWindows向けツールです。Windows 11 の新しい右クリックメニューと「その他のオプション」の両方に対応します。

定型コピー

毎回EXIFビューアーを開かず、必要な情報だけを定型フォーマットでコピーできます。

特徴

右クリックから完結

写真ファイルを右クリックし、「EXIF情報をコピー」から使いたい形式を選ぶだけでコピーできます。

投稿向けに整形

カメラ名、レンズ名、焦点距離、F値、シャッタースピード、ISO感度などを見やすい順で並べてコピーできます。

フォーマットも編集可能

よく使う並びに合わせて、設定画面から出力フォーマットを自由に調整できます。

対応ファイル形式

右クリック表示

拡張子で絞り込まず、Windowsの右クリックメニューに表示します。

主な想定ファイル

JPEG / PNG / TIFF / HEIC / WebP、ARW / CR2 / CR3 / NEF / RAF / ORF / RW2 / DNG など。

EXIF取得について

ファイルや環境によって取得できる項目は変わります。取得できた項目だけを使ってコピーします。

機能

1

右クリックで EXIF コピー

写真ファイルを右クリックし、「EXIF情報をコピー」から使いたい形式を選ぶと、カメラ名・レンズ名・焦点距離・F値・ISO感度などを整形してコピーできます。

2

フォーマット追加・編集

SNS向け、ブログ向け、記録用など、自分の使い方に合わせてフォーマットを追加したり、表示する項目や並び順を編集したりできます。
