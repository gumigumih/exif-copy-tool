写真のEXIF情報を、右クリックからすばやくコピーできるWindows向けツールです。

インストーラーをダウンロード リリースを見る

右クリック → EXIF情報をコピー → SNS用

こんな方におすすめ

SNS投稿

X（Twitter）やDiscordで撮影設定を添えて写真を共有したい方に。

ブログ執筆

レビュー記事や撮影記録にカメラ・レンズ情報を掲載したい方に。

定型コピー

毎回EXIFビューアーを開かず、必要な情報だけを定型フォーマットでコピーできます。

特徴

右クリックですぐコピー

画像ファイルを右クリックするだけで、登録したフォーマットでEXIF情報をコピーできます。

自由にカスタマイズ

撮影日時、カメラ、レンズ、ISO感度、シャッタースピードなどを自由に組み合わせられます。

Windows専用

Windows 10 / Windows 11 で使えます。インストール後すぐ右クリックメニューから利用できます。

対応ファイル形式

一般的な画像

JPEG / PNG / TIFF / HEIC / WebP に対応しています。

RAWファイル

ARW / CR2 / CR3 / NEF / RAF / ORF / RW2 / DNG などから、取得できる範囲のEXIF情報を読み取ります。

その他のファイル

EXIF情報がないファイルでは、取得できる項目のみを使ってコピーします。

使い方

  1. インストーラーをダウンロードして実行します。
  2. 写真ファイルを右クリックし、「EXIF情報をコピー」からフォーマットを選びます。
  3. 整形されたEXIF情報がクリップボードへコピーされます。

配布情報

インストール先は %LOCALAPPDATA%\Programs\ExifCopyTool です。Pythonは不要です。アンインストール時に右クリックメニューも解除します。
